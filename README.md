## 📖 **About The Project**

A beautiful and intuitive weather application that provides real-time weather information for any city worldwide. Built with vanilla JavaScript and the OpenWeatherMap API, this app offers a clean, modern interface to display current weather conditions, including temperature, humidity, wind speed, and more.

### **Why I Built This**
- To practice working with third-party APIs
- To improve my JavaScript async/await skills
- To create a useful tool with a beautiful UI
- To understand weather data visualization

---

## ✨ **Features**

| Feature | Description |
|---------|-------------|
| 🌍 **Global Search** | Search weather for any city worldwide |
| 🌡️ **Current Temperature** | Displays temperature in Celsius |
| 💨 **Wind Speed** | Real-time wind speed in km/h |
| 💧 **Humidity** | Current humidity percentage |
| 👁️ **Visibility** | Visibility distance in kilometers |
| 📊 **Pressure** | Atmospheric pressure in mb |
| 🌡️ **Feels Like** | Perceived temperature |
| 💧 **Dew Point** | Calculated dew point temperature |
| 🎨 **Weather Icons** | Visual representation of weather conditions |
| 📝 **Description** | Text description of weather (sunny, cloudy, etc.) |
| 🎯 **Responsive Design** | Works perfectly on all devices |
